# 進階 2D 幾何 Tutting 舞蹈模擬器 (專業七頭身站姿版)

本專案是一款專為 **Tutting (埃及手) 街舞愛好者**與**空間幾何視覺設計師**打造的極簡主義風格火柴人模擬器。它能輔助使用者進行編舞、空間方框疊加（Box Folding）設計與動作練習。

本工具完全使用 **原生 HTML5 Canvas、CSS3 與 Vanilla JavaScript** 實作，無任何外部依賴（如 Three.js, p5.js 或 GSAP），具備極高的流暢度與跨平台相容性。

## 🎯 核心功能亮點

1. **極簡高對比視覺與 RWD 設計**
   - 採用深色調高冷時尚風格（背景 `#08080a`，骨架純白 `#ffffff`）。
   - 設有極暗的十字幾何基準線（`#111116`），完美輔助直角與方框對齊。
   - 響應式佈局（RWD），針對行動端觸控進行優化，關節判定半徑擴大至 `26px`，按鈕點擊區域均符合 `44x44px` 規範。

2. **真實系「七頭身」黃金幾何比例**
   - 嚴格遵循數學計算消除卡通感（總身高為頭高的 7 倍，即 210px）。
   - 大臂與前臂採 `50:50` 等長、大腿與小腿採 `55:55` 等長。確保在 180° 折疊或極致深蹲時線條完美重合，達成極致的「箱型摺疊（Box Folding）」效果。

3. **全身雙模式互動系統 (FK / IK)**
   - **FK (正向運動學) 模式**：可獨立拖曳 12 個核心關節，子關節會隨父階層產生正確的矩陣旋轉連動。
   - **IK (逆向運動學) 模式**：使用 **FABRIK** 演算法。拖曳手腕或腳踝時，自動反向解算大臂/前臂（或大腿/小腿）角度。具備**邊界安全鎖**，當拉扯超出最大範圍時自動鎖定，防止骨骼脫臼分離。

4. **完美 Tutting 角度吸附機制 (Angle Snapping)**
   - 當關節角度接近 `0°`, `45°`, `90°`, `135°`, `180°`, `225°`, `270°`, `315°` 等 45 度倍數時（容許誤差 `±5°`），系統會自動精準吸附（Snap）。
   - 觸發時關節會轉為螢光青綠色（`#00ffcc`）並擴大閃爍，狀態列同步切換為 `TRIGGERED`。

5. **Time-based LERP 動畫時間軸引擎**
   - 支援動態「新增/刪除/清除」關鍵影格，並可循環播放（Loop）。
   - 動畫基於時間差（Delta Time）平滑計算，不受瀏覽器幀率影響。
   - **臨界點優化**：採用最短路徑角度內插演算法（利用餘弦與正弦處理差值），嚴格防止四肢在跨越 `+180°` 與 `-180°` 邊界時發生瘋狂自轉的現象。

## 🛠️ 技術規格
- **開發語言**：HTML5, CSS3, JavaScript (ES6+)
- **核心演算法**：FABRIK (Forward And Backward Reaching Inverse Kinematics), Shortest-path Angular Interpolation
- **軟體架構**：100% 獨立純原生代碼，無外部依賴。

## 🚀 快速上手

1. 將模擬器的 HTML 程式碼儲存為 `tutting_simulator.html`。
2. 使用任何現代瀏覽器（Chrome, Safari, Edge 等）直接雙擊打開該檔案。
3. **FK 模式**：點擊並拖曳白色的關節點（如手肘、手腕），調整細部相對角度。
4. **IK 模式**：切換至 IK 模式，直接拖曳手腕或腳踝末端，感受流暢的反向動力學跟隨。
5. **編舞時間軸**：
   - 擺好姿勢後，點擊 **「＋ 新增影格」** 儲存當前動作。
   - 調整下一個姿勢，再次點擊新增。
   - 點擊 **「▶ PLAY」** 即可觀看自動平滑過渡的 Tutting 舞蹈動畫。
